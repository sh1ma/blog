# gh CLI

`sh1ma/blog` に対する `gh` コマンド操作のルートスキル。用途別にサブスキルへ分かれている。

## サブスキル

- [`gh:pr`](./pr/SKILL.md) — PR (プルリクエスト) の作成・レビュー・マージ・コメント確認。

## 共通ルール

- **対象は `sh1ma/blog` のみ**。`--repo` で他リポジトリを指定しない。
- **具体的なフラグは実行時に `gh <sub> --help` で確認する**。このスキル群には CLI の使い方を書かない。
