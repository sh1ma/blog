Whatever happened to my goal of "writing one blog post a month"...? I didn't manage to write anything in September. But I'd like to run with the theory that if I write two posts this month, we're good. I'll do it. So — this is post number one for October.

Before I knew it, the heat was gone and the sun started setting earlier. A friend told me, "It smells like osmanthus." So maybe autumn really has started. It made me try to remember whether autumn felt like this last year too, but I couldn't really recall.

Between August and October I read a few Akutagawa Prize winners:

- Bari Sanko
- Black Box
- Idol, Burning
- Tokyo-to Doujou-tou

That's four books. I can't really give impressions — or rather, I read them a while ago and I don't remember well. None of them were unpleasant to read.  
Akutagawa Prize winners are easy to read, and so far I haven't hit one that felt like a miss, so I've been happily working through them. Right now I'm partway through a book called *Kono Yo no Yorokobi Yo* ("The Joys of This World"). My goal for the year is to read 10 books, so I've got a few more to go. Fingers crossed.

## Wrapping Up

I've got nothing to write. There should've been plenty of events in my life, but I don't remember any of them. This kind of thing is why I want to start keeping a record of daily life so I can look back on it.

Oh, right — I bought an iPhone 16 and a 10th-generation Apple Watch. That's about all I've got.

I'll write one more post this month. For real.
